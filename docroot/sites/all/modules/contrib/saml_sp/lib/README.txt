SAML Service Provider - PHP-SAML library
========================================

Install the OneLogin SAML-PHP toolkit to this directory:

`cd lib`
`git clone https://github.com/onelogin/php-saml.git .`


Directory structure
===================

You should see:
- saml_sp
  - lib
    - ext
      - xmlseclibs
        - xmlseclibs.php
    - README.txt
    - src
      - OneLogin
        - Saml
          - AuthRequest.php
          - Metadata.php
          - Response.php
          - Settings.php
          - XmlSec.php
  - README.txt
  - saml_sp.admin.inc
  - saml_sp.info
  - saml_sp.module
  - saml_sp.pages.inc
